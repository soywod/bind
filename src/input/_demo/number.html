Inputs

Enhanced input type number. Value is formatted on blur via Intl.NumberFormat.

Attributes

 Name Description Usage Example min Minimum value allowed. max Maximum value allowed. step Default step used on increment and decrement. locale Intl locale used to format the value. type
Intl style. Possible values: decimal (default), percent, currency.
currency
Intl currency. The type should be set to currency. Possible values: see currency-iso.org.

Properties

 Name Description Usage Example intl
Custom Intl.NumberFormat instance.
const elem = document.querySelector("cx-input-number"); elem.intl = new Intl.NumberFormat("en-US", {style: "currency", currency: "CNY"});
